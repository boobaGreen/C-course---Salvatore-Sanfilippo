## Usare Typedef

`typedef` non crea un nuovo tipo, ma un "soprannome" per un tipo esistente. Salvatore lo consiglia caldamente per evitare di scrivere `struct` ogni volta.

```c
typedef struct {
    int x;
    int y;
} Point;

Point p; // Molto più pulito di 'struct Point p'
```

<Callout type="info" title="Perché non usare sempre typedef?">
Salvatore osserva che, sebbene sia comodo, abusare di `typedef` per i tipi base (es. `typedef int error_code`) può nascondere la natura del dato e rendere il codice più difficile da leggere per chi non conosce i tuoi alias. Usalo con saggezza!
</Callout>

### Struct Anonime
Notate che nella `typedef` sopra non abbiamo dato un nome alla struct dopo la parola chiave `struct`. È una **Struct Anonima**: ci serve solo il nome dell'alias `Point` per usarla nel programma.

## L'API dei File (stdio.h)

In C, un file è visto come uno **stream** di byte. La libreria standard ci fornisce un puntatore a una struttura speciale chiamata `FILE`.

<Callout type="hacker" title="Tipi Opachi">
Il tipo `FILE` è il primo esempio di **Tipo Opaco** che incontriamo. Noi non sappiamo (e non dobbiamo sapere) cosa c'è dentro la struttura `FILE`. Usiamo solo il puntatore e le funzioni (`fopen`, `fread`, ecc.) per manipolarlo. Questo è l'eccellenza del design: nascondere la complessità interna.
</Callout>

<Callout type="info" title="Il Vantaggio dell'Opacità">
  Usando tipi opachi, gli sviluppatori della libreria C possono cambiare completamente come funziona `FILE` (magari per renderlo più veloce) senza che tu debba cambiare una singola riga del tuo programma. È il principio del **minimo privilegio**: meno sai di come è fatta una cosa, meno puoi romperla.
</Callout>

### Esempio: Scrivere e Leggere un Binario

<CodeEditor 
  initialCode={`#include <stdio.h>
#include <stdint.h>

typedef struct {
    uint32_t id;
    float score;
} Player;

int main() {
    Player p1 = {1, 95.5f};
    
    // Scrittura
    FILE *fout = fopen("save.dat", "wb");
    if (fout) {
        fwrite(&p1, sizeof(Player), 1, fout);
        fclose(fout);
    }
    
    // Lettura
    Player loaded;
    FILE *fin = fopen("save.dat", "rb");
    if (fin) {
        fread(&loaded, sizeof(Player), 1, fin);
        printf("ID: %u, Score: %.1f\\n", loaded.id, loaded.score);
        fclose(fin);
    }
    
    return 0;
}`}
