## Prendere Decisioni: if ed else

Il cuore della logica di programmazione è il bivio. Se succede X, fai Y, altrimenti fai Z. 

<Callout type="info" title="Cosa è Vero in C?">
In C non esiste un tipo `bool` nativo (almeno nelle versioni classiche). Per il C:
- **0** è Falso.
- **Qualsiasi altra cosa** (1, -5, 42) è Vero.
Questo è il motivo per cui puoi scrivere `if (count)` per controllare se un contatore non è zero!
</Callout>

## Il Potere delle Graffe `{}`

In C, puoi aprire un blocco di codice ovunque, non solo dopo un `if`. Ogni blocco crea un nuovo livello di esistenza per le variabili.

<ScopeExplorer />

<Callout type="warning" title="Variabili Fantasma">
Se dichiari `int x = 5;` dentro un blocco, una volta uscito da quel blocco `x` non esiste più per il compilatore. Tentare di usarlo causerà un errore!
</Callout>

## L'Oscuro Goto

Salvatore ci mostra anche il `goto`, un modo per saltare direttamente a un'etichetta nel codice. Sebbene potente (e alla base di come il processore esegue i loop), è considerato pericoloso perché può rendere il codice difficile da seguire ("Spaghetti Code").

<CodeEditor 
  initialCode={`#include <stdio.h>\n\nint main() {\n    int i = 0;\n    loop:\n    printf("%d ", i);\n    i++;\n    if (i < 5) goto loop;\n    return 0;\n}`}
