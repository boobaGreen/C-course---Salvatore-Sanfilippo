## Il Vincolo come Opportunità

Lo ZX Spectrum aveva una gestione della memoria video estremamente creativa per risparmiare RAM. Con soli **6.75 KB** riusciva a gestire uno schermo da 256x192 pixel. 

### La Mappa della Memoria
La memoria non è sequenziale: i pixel sono in una zona, i colori (attributi) in un'altra.

<SpectrumMemoryVisualizer />

---

## Il Problema degli Attributi (Attribute Clash)

A differenza dei computer moderni, dove ogni pixel può avere il suo colore, lo Spectrum assegna i colori a blocchi **8x8**. Ogni blocco ha un solo byte che definisce tutto:

<AttributeGrid />

<ZXPalette />

<Callout type="warning" title="Limiti Hardware">
In un blocco 8x8 puoi avere solo **due colori**: uno per lo sfondo (PAPER) e uno per i pixel accesi (INK). Se un personaggio colorato passa sopra uno sfondo di un altro colore, i colori "si scontrano" (clash).
</Callout>

---

## Algoritmi Evolutivi: Simulated Annealing

Come possiamo trasformare una foto moderna (PNG) in un'immagine che rispetti questi limiti brutali? Salvatore usa un **Algoritmo Evolutivo**.

1. **Mutazione**: Cambiamo a caso alcuni bit nella memoria dello Spectrum.
2. **Rendering**: Trasformiamo quella memoria in un'immagine.
3. **Fitness**: Confrontiamo l'immagine ottenuta con l'originale.
4. **Selezione**: Se la nuova immagine è "più vicina" all'originale, la teniamo.

<EvoSimulator />

### Il Curriculum Learning
Invece di ottimizzare tutto subito, Salvatore usa una strategia a fasi:
- **Fase 1**: Ottimizza solo i pixel (bianco e nero).
- **Fase 2**: Fissa i pixel e ottimizza i colori.
- **Fase 3**: Ottimizza tutto insieme per i ritocchi fini.

## Il Codice: Calcolare la Distanza

La funzione `compute_diff` è il cuore del programma. Tratta i colori RGB come coordinate in uno spazio 3D e calcola la distanza euclidea.

<CodeEditor 
  initialCode={`long long compute_diff(unsigned char *a, unsigned char *b) {
    long long diff = 0;
    for (int i = 0; i < WIDTH * HEIGHT * 3; i += 3) {
        int r = a[i] - b[i];
        int g = a[i+1] - b[i+1];
        int b = a[i+2] - b[i+2];
        // Distanza quadratica (più veloce della radice)
        diff += (r*r) + (g*g) + (b*b);
    }
    return diff;
}`}
  language="c"
