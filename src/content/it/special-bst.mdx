## Cos'è un Albero Binario di Ricerca?

Un **Binary Search Tree** è una struttura dati dove ogni nodo ha al massimo due figli (sinistro e destro). La proprietà fondamentale è:
- I valori a **sinistra** sono sempre **minori** del nodo attuale.
- I valori a **destra** sono sempre **maggiori** del nodo attuale.

## L'Implementazione in C

Salvatore definisce la struttura base in modo molto pulito:

<CodeEditor 
  initialCode={`struct node {
    int val;
    struct node *left;
    struct node *right;
};

// Creazione di un nuovo nodo
struct node *create_node(int val) {
    struct node *n = malloc(sizeof(struct node));
    n->val = val;
    n->left = n->right = NULL;
    return n;
}`}
  language="c"
/>

### L'Inserimento (Iterativo)

Per inserire un valore, "scendiamo" nell'albero confrontando il valore con ogni nodo finché non troviamo un posto vuoto (`NULL`).

<CodeEditor 
  initialCode={`struct node *insert(struct node *root, int val) {
    if (root == NULL) return create_node(val);
    struct node *curr = root;
    while (1) {
        if (val < curr->val) {
            if (curr->left == NULL) {
                curr->left = create_node(val);
                break;
            }
            curr = curr->left;
        } else {
            if (curr->right == NULL) {
                curr->right = create_node(val);
                break;
            }
            curr = curr->right;
        }
    }
    return root;
}`}
  language="c"
/>

## L'Eleganza della Ricorsione: Stampa Ordinata

Il momento più magico della lezione è quando Salvatore mostra come stampare i valori dell'albero dal più piccolo al più grande con pochissime righe di codice. Si chiama **In-order Traversal**.

<CodeEditor 
  initialCode={`void print_sorted(struct node *root) {
    if (root == NULL) return;
    
    print_sorted(root->left);   // 1. Vai a sinistra (valori minori)
    printf("%d ", root->val);   // 2. Stampa il nodo attuale
    print_sorted(root->right);  // 3. Vai a destra (valori maggiori)
}`}
  language="c"
/>

<Callout type="info" title="L'algoritmo emerge dalla struttura">
Salvatore sottolinea come non serva imparare il codice a memoria: se capisci che i valori piccoli sono a sinistra, la funzione ricorsiva "si scrive da sola". È l'eleganza dell'informatica.
</Callout>
