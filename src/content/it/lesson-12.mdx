## Oltre il Limite dello Stack

Fino ad ora, ogni variabile che abbiamo creato viveva nello **Stack**. Lo stack è veloce, sicuro, ma ha due grandi difetti:
1. Devi sapere la dimensione delle variabili **al momento della compilazione**.
2. Quando una funzione termina, **tutto il suo stack viene distrutto**.

Se vogliamo creare dati che "sopravvivono" alle funzioni, o dati la cui grandezza la sappiamo solo mentre il programma gira (es. un file testuale caricato dall'utente), dobbiamo usare l'**Heap**.

### Malloc e Free

Per accedere allo Heap si usano funzioni della libreria standard `<stdlib.h>`:

```c
#include <stdlib.h>

// Voglio spazio per 100 interi (100 * 4 byte = 400 byte)
int *numeri = malloc(100 * sizeof(int));

if (numeri == NULL) {
    // RAM Piena! Il SO ha detto no.
    return 1; 
}

// ... uso i numeri ...

// QUANDO HO FINITO, DEVO RESTITUIRE LA MEMORIA!
free(numeri);
```

<Callout type="warning" title="Usa sempre sizeof()">
Non scrivere mai `malloc(400)`. Usa `malloc(100 * sizeof(int))`. Il compilatore sa esattamente quanti byte occupa un `int` su quella macchina specifica (potrebbe essere 2, 4 o 8 byte).
</Callout>

## Evoluzione delle Stringhe

Nella lezione precedente abbiamo visto le *Pascal Strings* (lunghezza in un byte, max 255 caratteri).
Ora che abbiamo `malloc`, possiamo creare un **Oggetto Stringa** serio.

Il nostro formato personalizzato sarà:
1. I primi **4 byte** (`int`) conterranno la lunghezza della stringa.
2. I restanti byte conterranno i caratteri, più uno `\0` finale (per renderla compatibile con i comandi come `printf`).

### Creazione di una stringa ad alte prestazioni

<CodeEditor 
  initialCode={`#include <stdio.h>
#include <stdlib.h>
#include <string.h>

// Funzione per creare la nostra Super Stringa
char *crea_stringa(char *testo) {
    int lunghezza = strlen(testo);
    
    // Alloco: 4 byte per l'int + lunghezza del testo + 1 byte per '\0'
    char *nuova = malloc(sizeof(int) + lunghezza + 1);
    
    if (nuova == NULL) return NULL;
    
    // 1. Scrivo la lunghezza nei primi 4 byte usando il casting
    *((int *)nuova) = lunghezza;
    
    // 2. Copio il testo nei byte successivi
    char *inizio_testo = nuova + sizeof(int);
    strcpy(inizio_testo, testo);
    
    return inizio_testo; // Ritorno il puntatore al TESTO, non alla lunghezza!
}

// Funzione per leggere istantaneamente la lunghezza in O(1)
int lunghezza_v2(char *stringa_v2) {
    // Faccio un passo indietro di 4 byte e leggo l'intero
    return *((int *)(stringa_v2 - sizeof(int)));
}

int main() {
    char *s = crea_stringa("Ciao Hacker!");
    
    printf("Testo: %s\\n", s);
    printf("Lunghezza istantanea: %d\\n", lunghezza_v2(s));
    
    // Attenzione: free deve ricevere il puntatore originale!
    free(s - sizeof(int));
    
    return 0;
}`} 
  language="c"
