## La verità dietro i caratteri

Quando scrivi `char c = 'A';`, il computer non salva la lettera "A", ma il numero **65**. Questo è il cuore del sistema ASCII.

<Callout type="info" title="Apici vs Virgolette">
In C c'è una grande differenza:
- `'A'` (apici singoli) è un tipo `char` (un singolo numero).
- `"A"` (virgolette) è una stringa (un array di caratteri terminato da uno zero).
</Callout>

## Promozione dei Tipi

Sapevi che se passi un `char` a una funzione con argomenti variabili (come `printf`), il C lo promuove automaticamente a `int`? Ecco perché `%d` funziona anche per stampare il codice numerico di un carattere!

  initialCode={`#include <stdio.h>\n\nint main() {\n    char lettera = 'A';\n    printf("Carattere: %c\\n", lettera);\n    printf("Codice ASCII: %d\\n", lettera);\n    return 0;\n}`}
