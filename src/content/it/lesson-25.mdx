## Visualizzare gli Oggetti: La Potenza della Ricorsione

Prima di leggere, dobbiamo saper scrivere. Una lista ToyFort può contenere altre liste. Per stamparle correttamente, la nostra funzione `print_object` deve essere ricorsiva.

<CodeEditor 
  initialCode={`void print_object(TFObject *o) {
    switch(o->type) {
        case TF_INT:
            printf("%lld", o->i);
            break;
        case TF_LIST:
            printf("[ ");
            for (size_t i = 0; i < o->l.len; i++) {
                print_object(o->l.ele[i]); // Chiamata RICORSIVA
                if (i < o->l.len - 1) printf(" ");
            }
            printf(" ]");
            break;
        case TF_SYM:
            printf("%s", o->s.ptr);
            break;
    }
}`}
  language="c"
/>

<Callout type="warning" title="Il segreto del Newline">
Salvatore nota che senza un `\\n` finale, la shell (come la Z-shell o `zsh`) potrebbe mostrare un carattere `%` o invertito alla fine dell'output. Per questo aggiungiamo un `printf("\\n")` solo nel chiamante principale, non dentro la funzione ricorsiva!
</Callout>

<Callout type="info" title="Perché la Ricorsione?">
Non è un esercizio accademico! La ricorsione è il modo più naturale per navigare strutture "ad albero" come i programmi. Senza di essa, dovremmo gestire manualmente uno stack complicatissimo.
</Callout>

<Callout type="warning" title="Limite della Ricorsione">
  Attenzione: se un utente malintenzionato crea una lista annidata 1 milione di volte (es. `[[[[...]]]]`), la funzione `print_object` esaurirà lo spazio della RAM dello stack della CPU (Stack Overflow). Salvatore consiglia di mettere sempre un contatore di profondità massima nelle funzioni ricorsive.
</Callout>

## Implementare il Parser di Simboli

Un simbolo in ToyFort è qualsiasi sequenza di caratteri che non sia un numero o una stringa. Dobbiamo definire un set di caratteri accettabili (alfanumerici + simboli matematici).

### La funzione `is_symbol_char`

Salvatore usa un trucco elegante con `strchr` per verificare se un carattere appartiene a un set di simboli ammessi:

```c
int is_symbol_char(int c) {
    if (isalpha(c)) return 1;
    if (strchr("+-*/%=<>!?", c)) return 1;
    return 0;
}
```
