## Generare la Sorte: `rand()` e `srand()`

In C, la funzione `rand()` genera un numero **pseudocasuale** tra 0 e `RAND_MAX`. 

### Il Problema del Seme (Seed)

Se lanci un programma che usa `rand()` più volte, otterrai **sempre gli stessi numeri**. Questo perché il computer segue un algoritmo deterministico. Per cambiare serie, dobbiamo "piantare un seme" diverso ogni volta usando `srand()`.

<CodeEditor 
  initialCode={`#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    // Inizializza il seme con l'ora attuale
    srand(time(NULL));
    
    // Genera un numero tra 0 e 9
    int random_num = rand() % 10;
    
    printf("Numero fortunato: %d\\n", random_num);
    return 0;
}`}
  language="c"
/>

<Callout type="warning" title="Non per la Sicurezza!">
`rand()` non è adatto per la crittografia. Se devi generare chiavi di sicurezza, servono librerie specializzate o file di sistema come `/dev/urandom`.
</Callout>
