## Puntatori a Puntatori

Anche un puntatore è una variabile, e come tale ha un suo indirizzo di memoria. Possiamo dichiarare un "puntatore a un puntatore":
- `int x = 5;`
- `int *y = &x;` (y punta a x)
- `int **z = &y;` (z punta a y)

Tutti i puntatori (a `char`, `int`, o altri puntatori) hanno la **stessa dimensione** (8 byte su sistemi a 64 bit), perché contengono sempre un indirizzo di memoria.

## Stringhe e Puntatori

In C, le stringhe sono array di `char` terminati dal carattere speciale `\0` (null terminator). Possiamo scorrere una stringa incrementando un puntatore finché non incontriamo lo zero.

<CodeEditor 
  initialCode={`#include <stdio.h>

void stampa_stringa(char *p) {
    while (*p != '\\0') {
        putchar(*p); // Stampa il carattere corrente
        p++;         // Passa al prossimo byte
    }
    putchar('\\n');
}

int main() {
    char *saluto = "Hacker World";
    stampa_stringa(saluto);
    return 0;
}`}
  language="c"
/>

---

<KeyConcepts title="Hacker Key Concepts">
  - **Buffer Overflow**: Cosa succede quando scrivi oltre i limiti di un puntatore/array.
  - **Memory Layout**: Capire come i dati sono impacchettati (punti di forza del C).
  - **Little Endian**: Come i numeri multi-byte sono memorizzati (il byte meno significativo all'indirizzo più basso).
</KeyConcepts>
