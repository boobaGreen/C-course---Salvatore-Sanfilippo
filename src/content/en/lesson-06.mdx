## Making Decisions: if and else

The heart of programming logic is the crossroads. If X happens, do Y; otherwise, do Z.

<Callout type="info" title="What is True in C?">
In C, there is no native `bool` type (at least in classic versions). For C:
- **0** is False.
- **Anything else** (1, -5, 42) is True.
This is why you can write `if (count)` to check if a counter is non-zero!
</Callout>

## The Power of Braces `{}`

In C, you can open a block of code anywhere, not just after an `if`. Every block creates a new level of existence for variables.

<ScopeExplorer />

<Callout type="warning" title="Ghost Variables">
If you declare `int x = 5;` inside a block, once you exit that block `x` no longer exists for the compiler. Trying to use it will cause an error!
</Callout>

## The Dark Goto

Salvatore also shows us `goto`, a way to jump directly to a label in the code. While powerful (and the basis for how the processor runs loops), it's considered dangerous because it can make code hard to follow ("Spaghetti Code").
