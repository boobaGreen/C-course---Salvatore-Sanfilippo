## Implementing xdump

Salvatore shows us how to transform a `void *` pointer into a series of readable bytes. The goal is to print both the hexadecimal value and the ASCII character (if printable).

<CodeEditor 
  initialCode={`#include <stdio.h>
#include <string.h>
#include <ctype.h>

void xdump(void *p, int len) {
    unsigned char *c = p;
    for (int i = 0; i < len; i++) {
        // Ternary operator: if printable show char, otherwise a dot
        printf("%02x (%c) ", c[i], isprint(c[i]) ? c[i] : '.');
        if ((i + 1) % 8 == 0) printf("\\n");
    }
    printf("\\n");
}

## The Ternary Operator: `? :`

It is the "hacker" version of the if-else. Salvatore often uses it for quick decisions:
`condition ? value_if_true : value_if_false`

Example:
`char display = isprint(c) ? c : '.';`

## Memset: Deep Cleaning

In C, newly allocated memory or memory declared on the stack contains "garbage". With `memset`, we can zero everything out in one go.

### Final Summary
We have learned not to trust abstractions: memory is a tape of bytes and we now have the tools to inspect and manipulate it with surgical precision. `memset` and `xdump` will be your best friends in low-level debugging.
