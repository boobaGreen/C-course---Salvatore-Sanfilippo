# Lesson 30: Evolutionary Aesthetics and Ancient Hardware

In this "midnight" lesson, Salvatore takes us on a journey through time and memory space, exploring how C can manipulate bits to interface with 40-year-old hardware: the **ZX Spectrum**.

## Constraint as Opportunity

The ZX Spectrum had extremely creative video memory management to save RAM. With just **6.75 KB**, it could drive a 256x192 pixel display.

### The Memory Map
Memory is not sequential: pixels are in one area, colors (attributes) in another.

## The Attribute Problem (Attribute Clash)

Unlike modern computers where every pixel can have its own color, the Spectrum assigns colors in **8x8 blocks**. Each block has a single byte defining everything:

<AttributeGrid />

<ZXPalette />

<Callout type="warning" title="Hardware Limits">
In an 8x8 block, you can have only **two colors**: one for the background (PAPER) and one for active pixels (INK). If a colored character passes over a background of another color, the colors "clash."
</Callout>

---

## Evolutionary Algorithms: Simulated Annealing

How can we transform a modern photo (PNG) into an image that respects these brutal limits? Salvatore uses an **Evolutionary Algorithm**.

1. **Mutation**: Randomly change some bits in the Spectrum's memory.
2. **Rendering**: Transform that memory into an image.
3. **Fitness**: Compare the resulting image with the original.
4. **Selection**: If the new image is "closer" to the original, we keep it.

<EvoSimulator />

### Curriculum Learning
Instead of optimizing everything at once, Salvatore uses a phased strategy:
- **Phase 1**: Optimize pixels only (black and white).
- **Phase 2**: Fix pixels and optimize colors.
- **Phase 3**: Optimize everything together for fine-tuning.

---

## The Code: Calculating Distance

The `compute_diff` function is the heart of the program. It treats RGB colors as coordinates in 3D space and calculates the Euclidean distance.

### Measuring the Difference

To decide if a mutation is 'better', the algorithm calculates how much each Spectrum pixel differs from the original using Euclidean distance.

  Salvatore avoids using the square root (`sqrt`) in distance calculation: it's CPU-intensive and, for comparison purposes, `a² + b² + c²` is perfectly sufficient to determine which image is closer to the original.
