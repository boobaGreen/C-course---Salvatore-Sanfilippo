## Generating Luck: `rand()` and `srand()`

In C, the `rand()` function generates a **pseudorandom** number between 0 and `RAND_MAX`.

### The Seed Problem

If you run a program using `rand()` multiple times, you will get **the same numbers every time**. This is because the computer follows a deterministic algorithm. To change the sequence, we must "plant" a different seed each time using `srand()`.

<CodeEditor 
  initialCode={`#include <stdio.h>
#include <stdlib.h>
#include <time.h>

int main() {
    // Initialize the seed with the current time
    srand(time(NULL));
    
    // Generate a number between 0 and 9
    int random_num = rand() % 10;
    
    printf("Lucky number: %d\\n", random_num);
    return 0;
}`}
  language="c"
/>

<Callout type="warning" title="Not for Security!">
`rand()` is not suitable for cryptography. If you need to generate security keys, you need specialized libraries or system files like `/dev/urandom`.
</Callout>
