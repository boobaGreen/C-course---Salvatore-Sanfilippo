## The Truth Behind Characters

When you write `char c = 'A';`, the computer doesn't save the letter "A", but the number **65**. This is the core of the ASCII system.

<Callout type="info" title="Quotes vs Apostrophes">
In C, there's a big difference:
- `'A'` (single quotes) is a `char` type (a single number).
- `"A"` (double quotes) is a string (a zero-terminated array of characters).
</Callout>

## Type Promotion

Did you know that if you pass a `char` to a function with variable arguments (like `printf`), C automatically promotes it to an `int`? This is why `%d` works for printing the numeric code of a character too!

<KeyConcepts title="Hacker Key Concepts">
  - **Characters are numbers**: Every character (`char`) is actually an 8-bit integer.
  - **ASCII Table**: The universal map that assigns a unique number to every text symbol.
