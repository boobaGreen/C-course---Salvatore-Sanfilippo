## The Physics of Recursion: Stack Frames

When `print_object` calls itself to print an internal list, the CPU creates a new **Stack Frame**. This frame contains the return address, parameters, and local variables.

<Callout type="pro" title="The Stack Overflow Danger">
  The CPU's stack has a limited size (usually 8MB on Linux/Windows). If a
  ToyFort list is nested 100,000 times, the recursion will exhaust the stack,
  and the OS will kill the process with a **Segmentation Fault (Stack
  Overflow)**.
</Callout>

### Pointer Math: Atomic Speed

Instead of copying every character into a temporary buffer, Salvatore's parser calculates the length by subtracting two pointers: `len = p - start`. This avoids unnecessary allocations until the final object creation.
