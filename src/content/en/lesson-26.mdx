<Callout type="pro" title="Cache Miss Physics">
  Unlike linked lists (Lesson 15), our ToyFort lists are arrays of pointers.
  This means the object addresses are physically close in memory. The CPU can
  pre-load these addresses into the **L1 Cache**, making the execution loop
  incredibly smooth.
</Callout>

## Dynamic Memory Management: Retain & Release

In an interpreter, an object can belong simultaneously to a list, the stack, and a variable. Who should free it? The answer is: the last one to let it go.

### The Programmer's Contract

When you receive an object via `pop()`, you obtain **Ownership**.

- If you move it elsewhere: you must call `retain()`.
- When you are finished: you MUST call `release()`.

<Callout type="warning" title="Use of assert.h">
  Assertions like `assert(count > 0)` are fundamental. If the condition is false, the program aborts, indicating the file and line. It's the difference between debugging for hours and knowing instantly where the error is.
</Callout>
